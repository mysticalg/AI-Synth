Screenshots

Fresh captures from the standalone app, with one slide per major tab so each feature area reads clearly.

Voice Tab

Oscillator bank, filter tone shaping, arp and rhythm gate controls, plus the full-width performance keyboard in one compact layout.

Mod Tab

Dedicated ADSR slider banks for both envelopes, assignable velocity routing, and three LFOs ready for PWM, sync, and motion work.

FX / Matrix Tab

Host-synced delay plus chorus, distortion, saturation, compression, reverb, bitcrush, and four flexible modulation routing lanes.

Sequencer Tab

Pattern browser, host-sync or free-run timing, and per-step note, gate, and velocity lanes designed to stack with the arp for unusual movement.

What's Inside

Built for both local play and plugin-host workflows.

Synthesis

  • Multiple oscillators, sub, FM, filter drive, envelopes, and unison controls
  • Modulation matrix, LFOs, sequencing, and performance routing
  • Preset browser with grouped factory sounds

Playback

  • Standalone app for direct local use
  • VST3 build for plugin hosts
  • Host-synced timing plus free-running controls

Local Build

From the repo root:

powershell -ExecutionPolicy Bypass -File .\build-local.ps1 -Target All

The standalone app is written to build\AISynth_artefacts\Release\Standalone\AI Synth.exe.
